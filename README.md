# Sales Performance Analysis — SQL + Power BI

-- Objective
   Analysed 9,994 retail sales records to identify revenue trends,
   top-performing products, and loss-making anomalies across four regions.

-- Tools Used
    SQL (MySQL), Power BI, Excel

-- Key Insights
    West region generated the highest revenue (0.73M) across all regions 
    Technology category had the best profit ratio among all segments
     High discount orders (40% and above) consistently resulted in negative profit margins

      sales_analysis.sql — all 6 SQL queries used for analysis

-- Dataset
      Superstore Sales Dataset — Kaggle
